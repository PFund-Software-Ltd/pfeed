# Download

## Download Historical Data
```bash
pfeed download \
--data-source bybit
--product BTC_USDT_PERP
--resolution 1minute
--start-date 2025-01-01
--end-date 2025-01-02
```

> `pfeed download --help` for more options

```{seealso}
See [Supported Data Sources](../supported-data-sources.md) to find out what data sources are supported.
```
