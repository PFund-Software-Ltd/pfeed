## Using [Poetry](https://python-poetry.org) (Recommended)
```bash
poetry add pfeed

# update to the latest version:
poetry update pfeed
```


## Using Pip

```bash
pip install pfeed

# install the latest version:
pip install -U pfeed
```


## Checking your installation
```bash
$ pfeed --version

pfeed, version 0.0.1.dev4
```
